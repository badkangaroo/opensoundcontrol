Open Sound Control Library
by Mark Jones and Ed Lawrence

A .NET Library for the Open Sound Control format. This project's primary objective is to make an easy to use API for communicating with devices that use the OSC communication format. Specifically the authors are developing a hobby robot using the Make Controller. For more information about the Make Controller please visit the Making Things web site.

The project currently is under active development and no stable release is yet avaiable. If you are interested in contributing to this project please feel free to download the source. Patches can be submitted via email to Mark Jones.

More information about Open Sound Control can be found at http://opensoundcontrol.org/.
